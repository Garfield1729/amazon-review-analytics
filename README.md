# Amazon Review Analytics Pipeline

## Overview
This project implements an end-to-end Big Data pipeline to process Amazon product reviews using Spark and Airflow, followed by a machine learning model to perform sentiment analysis.

## Projects Included
1. Big Data Pipeline
   - Data cleaning and schema normalization
   - Large-scale processing using PySpark
   - Workflow orchestration using Airflow

2. Machine Learning
   - Sentiment analysis on cleaned review data
   - TF-IDF based feature extraction
   - Classification using Logistic Regression and Naive Bayes

## Tech Stack
- Linux
- Hadoop (HDFS)
- Apache Spark (PySpark)
- Apache Airflow
- Python, Scikit-learn, NLTK
- Tableau
- Git & GitHub

## Dataset
Amazon Product Reviews Dataset

## Goal
To demonstrate real-world data engineering and machine learning workflow with production-oriented project structure.
